    Introduction

    Welcome to the Umbrella API !

    Authentication

    Umbrella API uses API token to access for required authentication’s resources. Umbrella API expect API token to be included in all API requests to the server in request that look s like following:

    curl -H "Authorization: Bearer token“ -d "data”

    You must replace token with your personal API key.

    Get API Token

    This endpoint used to retrieve API token

    Sample request

    curl "http://localhost:8080/oauth/token"
  -X POST
  -u <clientId>:<clientSecret>
  -d "grant_type=password&username=<user_username>&password=<user_password>"

    Success response 

    {
  "access_token": "7f9471ff-5d7b-4aee-92ca-f13d9c66c37a",
  "token_type": "bearer",
  "refresh_token": "d5b09ce9-0a3a-404b-b754-fcea309256ac",
  "expires_in": 3599,
  "scope": "write"
}

    Failed response

    {
  "timestamp": 1433945751615,
  "status": 401,
  "error": "Unauthorized",
  "message": "Bad credentials",
  "path": "/oauth/token"
}

    HTTP Request

    POST http://localhost:8080/oauth/token

    Parameters

    Parameter Description
    clientId Used in basic auth, fill with client id
    clientSecret Used in basic auth, fill with client secret
    grant_type Fill grant type with password
    username Fill with your username
    password Fill with your password

    Refresh Token

    This endpoint used to retrieve API token using refresh_token.

    Sample request

    curl "http://localhost:8080/oauth/token"
  -X POST
  -u <clientId>:<clientSecret>
  -d "grant_type=refresh_token&refresh_token=<refresh_token>"

    Errors

    The Umbrella API uses the following error codes:

    Error Code Meaning
    400 Bad Request – Your request sucks
    401 Unauthorized – Your API key is wrong
    403 Forbidden – The kitten requested is hidden for administrators only
    404 Not Found – The specified kitten could not be found
    405 Method Not Allowed – You tried to access a kitten with an invalid method
    406 Not Acceptable – You requested a format that isn’t json
    410 Gone – The kitten requested has been removed from our servers
    418 I’m a teapot
    429 Too Many Requests – You’re requesting too many kittens! Slow down!
    500 Internal Server Error – We had a problem with our server. Try again later.
    503 Service Unavailable – We’re temporarially offline for maintanance. Please try again later.
