Search Report

This endpoint retrieves reports by Query String.

Sample request

curl "http://localhost:8080/api/reports/search?query=reportDate:2015-05-08"
  -H "Authorization: Bearer <token>"

HTTP request

GET http://localhost:8080/api/reports/search?{query}

URL Parameters

Parameter Description Format
query Query String String
Format Query String:

[attributeName] [operations] [value] , [attributeName] [operations] [value], …

Available operations:
: means equal or like
> means greater than or equal
< means less than or equal

Example:
query=reportDate>2015-06-06,reportDate<2015-06-20,policyNumber:AA23
